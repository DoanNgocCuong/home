# Claude Opus Multi-Agent Setup Requirements

## Tổng quan
- **Mục tiêu**: Setup Claude Opus Multi-Agent cho Technical Writer
- **Loại setup**: Tất cả (Team Agents + Built-in Skills + MCP Servers)

---

## Các câu hỏi và câu trả lời

### Q1: Loại Multi Agent
**Câu hỏi**: Bạn muốn setup loại Multi Agent nào?
- Claude Team Agents
- Built-in Skills
- MCP Servers
- **Tất cả** ✓

---

### Q2: Team Agents
**Câu hỏi**: Bạn muốn tạo Team Agents như thế nào?

**Trả lời**:
> Mình đang cần 1 team để giúp mình hệ thống và viết các bài viết chuẩn TECHNICAL WRITER: chẳng hạn như làm research, viết report, viết cases study, phân tích sai lầm, .... (Mình chưa rõ như nào là chuẩn BEST PRACTICES, cần bạn giúp mình làm chuẩn best practices )

**Chi tiết yêu cầu**:
- Research (nghiên cứu)
- Viết report
- Viết case study
- Phân tích sai lầm
- Làm chuẩn Best Practices cho Technical Writing

---

### Q3: MCP Servers
**Câu hỏi**: Bạn cần kết nối MCP Servers nào?

**Trả lời**:
> Cần các MCP để check tài liệu, MCP hỗ trợ task chính về TECHNICAL WRITER MASTER

**Yêu cầu**:
- MCP để check tài liệu
- MCP hỗ trợ Technical Writing

---

### Q4: Nơi lưu tài liệu
**Câu hỏi**: Nơi lưu tài liệu/bài viết của bạn ở đâu?

**Trả lời**: Thư mục hiện tại

---

### Q5: Tìm kiếm từ internet
**Câu hỏi**: Bạn có cần tìm kiếm thông tin từ internet không?

**Trả lời**: Có (Web Search)

---

## Tóm tắt yêu cầu cuối cùng

| Thành phần | Yêu cầu |
|------------|---------|
| Team Agents | Technical Writer Team (Research, Report, Case Study, Best Practices) |
| MCP Servers | Check tài liệu, hỗ trợ Technical Writing |
| Workspace | Thư mục hiện tại |
| Web Search | Có |
