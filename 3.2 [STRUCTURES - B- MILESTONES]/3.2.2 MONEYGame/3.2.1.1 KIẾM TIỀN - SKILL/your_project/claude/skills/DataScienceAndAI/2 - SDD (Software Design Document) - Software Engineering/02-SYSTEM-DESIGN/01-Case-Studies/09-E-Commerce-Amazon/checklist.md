# Checklist - E-Commerce (Amazon)

## Mức độ cơ bản (Basic)
- [ ] E-commerce system core components?
- [ ] Product catalog: database schema?
- [ ] Shopping cart: store where, how?
- [ ] Inventory management: track levels?
- [ ] Order processing: steps from cart to shipment?
- [ ] Payment processing: security considerations?
- [ ] User accounts: authentication, addresses, payment methods?
- [ ] Search functionality: product search implementation?

## Mức độ trung bình (Intermediate)
- [ ] Cart operations: add, update, delete, checkout?
- [ ] Inventory consistency: ensure no overselling?
- [ ] Distributed transactions: handle multi-service operations?
- [ ] Product recommendations: collaborative vs content-based?
- [ ] Reviews and ratings: aggregate, display?
- [ ] Pricing: fixed vs dynamic pricing?
- [ ] Caching strategy: what to cache, invalidation?
- [ ] Order tracking: status updates, notifications?
- [ ] Returns: manage return process, refunds?

## Mức độ nâng cao (Advanced)
- [ ] Inventory sharding: partition by region, product type?
- [ ] Flash sales: handle inventory limits, surge traffic?
- [ ] Multi-seller marketplace: seller management?
- [ ] Fraud detection: prevent fraudulent transactions?
- [ ] Payment gateway integration: support multiple providers?
- [ ] Subscription models: Prime membership?
- [ ] Price optimization: dynamic pricing, supply/demand?
- [ ] Regional fulfillment: optimize shipping?
- [ ] Analytics: track user behavior, conversion?
- [ ] Disaster recovery: geo-redundancy?

## Thực hành
- [ ] Design product, user, order, inventory schemas
- [ ] Implement product search with filters
- [ ] Build shopping cart service
- [ ] Implement checkout flow with payment
- [ ] Design inventory management system
- [ ] Add recommendation engine
- [ ] Implement order tracking
- [ ] Build rating/review system
