# Checklist - Search Engine (Google)

## Mức độ cơ bản (Basic)
- [ ] Search engine components: crawl, index, query?
- [ ] Inverted index: data structure nào, cách build?
- [ ] Crawling: BFS vs DFS, which better?
- [ ] PageRank algorithm: idea là gì?
- [ ] TF-IDF: relevance scoring technique?
- [ ] Query processing: tokenization, stemming?
- [ ] Indexing challenges: storage, updates?
- [ ] Ranking signals: relevance, freshness, authority?

## Mức độ trung bình (Intermediate)
- [ ] Web crawler: URL frontier, politeness policy?
- [ ] Distributed crawling: how to partition work?
- [ ] Index compression: techniques to reduce size?
- [ ] Query latency: optimize < 100ms response?
- [ ] Caching: cache popular queries?
- [ ] Spell correction: edit distance, language model?
- [ ] Auto-complete: prefix matching, frequency-based?
- [ ] Link analysis: handle cycles, dangling links?
- [ ] Duplicate detection: Bloom filters, hashing?

## Mức độ nâng cao (Advanced)
- [ ] Ranking algorithms: BM25, learning to rank?
- [ ] Personalization: user history, behavior?
- [ ] Real-time indexing: handle fresh content?
- [ ] Distributed index: sharding strategy?
- [ ] Multi-language support: tokenization challenges?
- [ ] Spam detection: link spam, content spam?
- [ ] Featured snippets: extraction and ranking?
- [ ] Vertical search: specialized search types?
- [ ] User intent: interpret query meaning?
- [ ] Incremental updates: efficient index updates?

## Thực hành
- [ ] Design crawler with URL frontier
- [ ] Build inverted index from documents
- [ ] Implement basic ranking (TF-IDF)
- [ ] Implement query processing (tokenization, etc.)
- [ ] Optimize search latency with caching
- [ ] Design data structures for billions of pages
- [ ] Implement spell correction
- [ ] Add pagination to results
