## 7. Việc cần làm sau dọn dẹp

### 7.1 Cần restart máy để hoàn tất
- [ ] Xoá `C:\Program Files\Imou_en` (314 MB)
- [ ] Xoá `C:\Program Files\PDFsam Enhanced 7` (306 MB)
- [ ] Xoá `C:\Program Files\MySQL` (690 MB)

### 7.2 Lưu ý khi sử dụng
- **JetBrains IDE** (WebStorm, PyCharm, DataGrip): Lần mở đầu tiên sẽ chậm hơn do tạo lại cache
- **GitKraken**: Cần đăng nhập lại, cache sẽ tự tạo lại
- **Lark**: Cache sẽ tự tải lại khi mở app
- **VS Code**: Settings giữ nguyên (sync qua account), extensions vẫn còn, chỉ mất cache
- **npm**: Cần cài lại global packages nếu cần (`npm install -g <package>`)
- **NuGet**: Tự tải lại packages khi build .NET project
- **pip**: Tự tải lại khi `pip install`
- **uv**: Tự tải lại khi `uv pip install`
- **Playwright**: Cần chạy `npx playwright install` nếu cần test browser
- **Ollama**: Cần tải lại model nếu muốn dùng (`ollama pull <model>`)

### 7.3 Genspark Chrome PWA còn sót
Genspark được cài dưới dạng Chrome PWA (Progressive Web App), registry entry vẫn còn nhưng không chiếm dung lượng đáng kể. Để xoá hoàn toàn:
1. Mở Chrome
2. Vào `chrome://apps`
3. Chuột phải vào Genspark > Remove from Chrome
