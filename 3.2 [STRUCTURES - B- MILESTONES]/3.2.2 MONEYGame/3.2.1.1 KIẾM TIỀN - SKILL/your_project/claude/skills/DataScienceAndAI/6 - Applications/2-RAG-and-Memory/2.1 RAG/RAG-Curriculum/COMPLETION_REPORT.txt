PRACTICAL RECOMMENDATIONS
==========================

Quick Start Stack:
├─ Parser: PyMuPDF (simple) or Unstructured (complex)
├─ Chunker: Recursive 512 tokens, 20% overlap
├─ Embedder: BGE-M3 (open-source) or OpenAI (quality)
├─ Vector DB: Qdrant (self-hosted) or Pinecone (managed)
├─ Framework: LangChain (flexibility) or LlamaIndex (data)
└─ Metadata: NER + keywords (80% of benefit)

For Vietnamese Projects:
├─ Framework: LlamaIndex (best Vietnamese support)
├─ Embedder: BGE-M3 (100+ languages including Vietnamese)
├─ Parser: Handle UTF-8 explicitly
├─ NER: spacy.load("vi_core_news_sm")
└─ All tools support Vietnamese out of the box

Cost per 100K documents (monthly):
├─ BGE-M3 + Qdrant self-hosted: ~$10 (compute) + $0 (storage)
├─ OpenAI + Pinecone: ~$25 (OpenAI) + $1 (Pinecone)
├─ Jina v3 + Qdrant: ~$5 (Jina embedding) + $10 (compute)
└─ LlamaIndex handles all frameworks seamlessly
